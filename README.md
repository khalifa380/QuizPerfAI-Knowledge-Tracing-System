# QuizPerfAI-Knowledge-Tracing-System
AI-powered Quiz-Based Knowledge Tracing System using GNN, GAN, Bayesian KT, SHAP Analysis, and Ensemble Learning for personalized student performance prediction and adaptive feedback.
Overview

QuizPerfAI is an AI-powered Quiz-Based Knowledge Tracing System developed to analyze student learning behavior and predict academic performance using multiple Machine Learning and Knowledge Tracing algorithms.

This project combines advanced AI techniques such as:

Graph Neural Networks (GNN)
Generative Adversarial Networks (GAN)
Bayesian Knowledge Tracing
Gradient Boosting
Monte Carlo SHAP Analysis
Ensemble Learning

The system provides:

Student performance prediction
Adaptive learning feedback
Domain-wise analysis
Privacy-preserving anonymization
Behavioral feature analysis

Features
Personalized student performance prediction
Adaptive feedback generation

SHAP-based explainable AI insights
Ensemble prediction system
GNN-based relationship learning
